:::info

**Setup** [Recommended]<br />
Installs NETworkManager on your system for all users (requires administrator privileges).<br />
Profiles and settings are stored in the user profile under `Documents\NETworkManager`.<br />

**Portable**<br />
Run NETworkManager anywhere without installation (e.g. USB stick, network share).<br />
Profiles and settings are stored in the same directory as the executable. Upgrades are supported by copying the `Profiles` and `Settings` directories to the newly downloaded version.

**Archive**<br />
Run NETworkManager anywhere without installation (e.g. USB stick, network share).<br />
Profiles and settings are stored in the user profile under `Documents\NETworkManager`. Might be useful for custom deployment senarios (e.g. Terminal Server, Intune, etc.).

See also the [system requirements](#system-requirements).

:::

## Package Manager

NETworkManager is available through the package managers [Chocolatey](https://chocolatey.org/packages/NETworkManager) and [WinGet](https://github.com/microsoft/winget-pkgs/tree/master/manifests/b/BornToBeRoot/NETworkManager/) and the PowerShell module [Evergreen](https://github.com/aaronparker/evergreen):

```
# Chocolatey
choco install networkmanager

# WinGet
winget install BornToBeRoot.NETworkManager

# Evergreen
Get-EvergreenApp -Name NETworkManager

# Evergreen (save to disk)
Get-EvergreenApp -Name NETworkManager | Save-EvergreenApp -Path C:\Users\$env:Username\Downloads\
```

## Silent install

The Setup is based on [InnoSetup](https://jrsoftware.org/isinfo.php) and you can use all available [command line parameters](https://jrsoftware.org/ishelp/index.php?topic=setupcmdline). Use the following parameters to perform a silent installation and create a desktop icon:

```
.\NETworkManager_{VERSION}_Setup.exe /VERYSILENT /SUPPRESSMSGBOXES /NORESTART /TASKS="desktopicon" /SP-
```
